Developed as a final project for my BSc at Birkbeck, this platform leverages Scala and Natural Language Processing (NLP) to autonomously organize and categorize large sets of documents.

Key features include:

- **Text Mining:** Automated extraction of key themes and entities.
- **Intelligent Classification:** Machine learning-based grouping of documents.
- **Scala-Native:** Built entirely within the Scala ecosystem to ensure performance and reliability.

Check out the source code on [GitHub](https://github.com/fredc0088/Intelligent_Documents_Organiser_Platform).
